== Description ==

Want to promote your website on web. What could better than marketing on facebook. Through Awesome Facebook Page Pluign you can easily  embed and promote your facebook page on your website. Simply like facebook you can share your page with your website visitors. Your Website visitors can like, share your page without having to leave your website.

= Backend editable fields from widget are: =

- Title
- Facebook Page URL
- Width
- Height
- Show/Hide Facebook Cover Image
- Show/Hide Facebook Facepile
- Show/Hide Posts
- Show/Hide Call to Action Button
- Enable Small Header
- Adaptive Container Width

We have setup default and best possible values in every fields so to get our widget start you just have to change the Facebook Page URL

== Installation ==

Installing the widget is very simple. You can download it via wordpress repository installer or download the plugin zip file and install it via wordpress installer.

Once you install it - make sure you have 'Activate' the plugin. After activation all you have to look at wordpress widget area and configure the plugin to your desire widget area.
